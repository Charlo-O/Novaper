目标是使用 Windows 记事本把输入文本保存到指定文件。

严格要求：
- 优先用 `launch_process`、`focus_window`、`uia_find`、`uia_set_value` 等结构化工具。
- 只有当 UIA 找不到控件或无法完成时，才使用 `computer` 工具执行点击、按键和视觉定位。
- 必须把文本完整保存到 `filePath`。
- 保存完成后，优先调用 `check_file` 读取文件确认内容，再结束任务。
- 如果出现安全弹窗、权限阻塞、UAC、验证码或无法跨越的系统边界，直接说明失败原因，不要编造成功。

执行建议：
1. 启动或聚焦 Notepad。
2. 在主编辑区写入输入文本。
3. 通过 `Ctrl+S` 触发保存。
4. 如需处理“另存为”对话框，填入精确文件路径并确认保存。
5. 用 `check_file` 校验文件存在且文本匹配。
